Objetivo
El objetivo de este repositorio es mostrar mi forma de abordar problemas de programación relacionados con complejidad algorítmica, optimización y paralelismo.

Además, quiero documentar el proceso de aprendizaje, incluyendo errores encontrados durante el desarrollo y la forma en que han sido analizados y corregidos.
