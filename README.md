# PEC4 – Programación para la Ciencia de Datos (UOC)

Este repositorio contiene la documentación del trabajo realizado en la **PEC 4**
de la asignatura Programación para la Ciencia de Datos** del
**Grado en Ciencia de Datos Aplicada (UOC).

Al igual que en la PEC anterior, he decidido documentar el proceso completo de trabajo,
desde las primeras aproximaciones hasta la solución final, con el objetivo de mostrar
de forma transparente cómo he abordado y resuelto cada uno de los ejercicios propuestos.


## Contenido del repositorio

- `README.md`: documentación general del proceso.
- Estructura del proyecto y organización por carpetas.
- Evidencias del desarrollo (capturas de pantalla, errores detectados y correcciones aplicadas).
- Referencias y material de apoyo consultado.


## Objetivo

El objetivo de este repositorio es mostrar mi forma de abordar problemas de programación
relacionados con modularidad, optimización y organización de un proyecto Python.

Además, se documenta el proceso de aprendizaje, incluyendo errores encontrados durante el
desarrollo y la forma en que han sido analizados y corregidos.


## Ejecución

Al tratarse de un trabajo entregable, he decidido no publicar el código completo
en GitHub. 


## Docstrings

Para esta PEC he utilizado docstrings en formato Google, siguiendo el mismo criterio
empleado en PECs anteriores.
